# barkometer
## Introduction
This repository contains the code used to build a bark-meter to monitor and register barking when out of the house.

The project uses a raspberry pi for collecting and presenting data. Data is captured using a VMA309 sound sensor.

## Roadmap
0) Setup hardware and verify working signal receieved by software.
1) Select analog or digital sound input. Tweak settings to register at a reasonable level.
2) Setup start and stop functions and write data to file.
4) Create a dashboard that presents the registered data.
5) Publish dashboard to make it accessible online.
